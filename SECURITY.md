# Security Policy

## Supported Versions

Currently, security updates are applied to the latest release and active main branch.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of Score Mate seriously. If you find a security vulnerability, please do not open a public issue. Instead, report it directly to the maintainers:

- **Email**: Send a detailed report to [your-email@example.com].
- **Details to Include**:
  - Description of the vulnerability.
  - Steps to reproduce (or a proof of concept).
  - Potential impact of the issue.

We will review your submission and aim to respond within 48 hours to confirm receipt and discuss a resolution timeline.
